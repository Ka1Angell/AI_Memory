# AI Memory — Схема

## Три слоя
| Слой | Назначение | Кто ведёт |
|------|------------|-----------|
| `Сырьё/` | Статьи, заметки, файлы (неизменяемое) | Ты |
| `Вики/` | Структурированные знания (растёт) | AI |
| `Схема/` | Правила, личность, контекст | Вместе |

## Сначала читать
`Схема/Личность` · `Схема/Контекст` · `Схема/Операции`

## Правила
- Извлекать из сессий → `Вики/`
- Обновлять `Контекст` при смене темы
- Решения → `Вики/Решения/`
- Фронтматтер: `type` · `status` · `created` · `updated`
- Синхронизация после каждой сессии
