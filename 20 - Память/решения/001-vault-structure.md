# Решение: Новая структура AI Memory vault

## Контекст
Старая структура была плоской (Разговоры, Мои действия, Задачи, Заметки). Не хватало:
- Структурированной памяти (профиль, контекст, глоссарий)
- Категоризации знаний (конфиги, софт, решения)
- AI-friendly точки входа (CLAUDE.md)

## Решение
Перейти на нумерованную PARA-подобную структуру:
- `10 - Входящие` — захват
- `20 - Память` — знания (AI ведёт)
- `30 - Сессии` — разговоры
- `40 - Действия` — лог AI
- `50 - Задачи` — todo

## Альтернативы
- Оставить плоскую структуру — отклонено (не масштабируется)
- PARA (Projects, Areas, Resources, Archives) — избыточно для текущего объёма

## Последствия
- Плагин авто-синхронизации обновлён под новые пути
- Старые папки остаются для обратной совместимости
