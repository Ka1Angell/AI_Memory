# Инструкции для AI

## Когда создаёшь заметку — всегда используй структуру:

### 1. Установка софта / Скачивание
```
# [Название]
## Требования
- ОС: ...
- Место: ...
## Подготовка
- [ ] Проверить систему
- [ ] Скачать установщик
## Установка
- Этап 1: ...
- Этап 2: ...
## Настройка
- [ ] ...
## Проверка
- [ ] Работает?
```

### 2. Настройка игры
```
# [Название игры]
## Требования
- Минимальные: ...
- Рекомендуемые: ...
## Моё железо
- CPU: ...
- GPU: ...
- RAM: ...
## Мои предпочтения
- Графика: ...
- FPS: ...
## Настройки
### Видео
- ...
### Управление
- ...
## Результат
- FPS: ...
- Стабильность: ...
```

### 3. Решение проблемы
```
# [Проблема]
## Симптомы
- ...
## Причина
- ...
## Решение
1. ...
2. ...
## Проверка
- [ ] Работает?
```

---

## Обязательно записывай в Obsidian:

### <thinking>
Твои размышления о задаче, варианты решения, сомнения
### </thinking>

### <answer>
Твой финальный ответ/решение
### </answer>

---

## Сохранение чатов

После каждой сессии:
1. Создай заметку в `Разговоры/YYYY-MM-DD - Сессия N.md`
2. Запиши суть разговора, решения, команды
