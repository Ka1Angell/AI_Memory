# Инструкции для AI

## Приоритеты
1. **Читать контекст** — в начале сессии: [[20 - Память/about.md]], [[20 - Память/working-context.md]], [[20 - Память/glossary.md]]
2. **Запоминать** — извлекать полезное из разговора в `20 - Память/`
3. **Синхронизировать** — после каждого ответа вызывать `obsidian-sync` + `obsidian-sync_syncGit`
4. **Обновлять контекст** — `working-context.md` при смене темы

## Шаблоны заметок

### Софт / Установка
```yaml
---
type: software
status: active | done | archived
---
```

### Конфиг / Настройки
```yaml
---
type: config
status: active | archived
---
```

### Решение
```yaml
---
type: decision
status: active | superseded
---
```

## Синхронизация
- Авто: OpenCode plugin → пишет сессии + действия
- Планировщик: ObsidianAISync → git push каждые 30 мин
- Ручная: скажи "синхронизируйся"
