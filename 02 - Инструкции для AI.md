# Инструкции

## Каждую сессию
1. Прочитать `⚙️ Схема/Личность` · `⚙️ Схема/Контекст` · `⚙️ Схема/Операции`
2. Извлечь новое → `🧠 Вики/`
3. На ответ → `obsidian-sync` + git sync

## Структура
```
📦 Сырьё/     → статьи, заметки, файлы (неизменяемое)
🧠 Вики/      → знания (AI пишет)
⚙️ Схема/     → правила (вместе)
💬 Сессии/    → чаты (авто)
📋 Лог/       → действия (авто)
📥 Входящие/  → захват
🎯 Задачи/    → todo
```

## Шаблон
```yaml
---
type: profile | config | decision | software | project | insight | term
status: active | done | archived
created: YYYY-MM-DD
---
```

## Стиль
- ✨ Использовать эмодзи в заметках
- 📝 Коротко и по делу
- 🔗 Связывать заметки ссылками
