# Инструкции

## Каждую сессию
1. Прочитать `Память/Обо мне` · `Память/Сейчас` · `Память/Словарь`
2. Извлечь новое → `Память/`
3. На ответ → `obsidian-sync` + git sync

## Структура
```
Входящие/   → быстрый захват
Память/     → знания (AI ведёт)
Сессии/     → чаты (авто)
Лог/        → действия AI (авто)
Цели/       → задачи
```

## Шаблон
```yaml
---
type: profile | config | decision | software | session | goal
status: active | done | archived
created: YYYY-MM-DD
---
```
