# Инструкции

## Каждую сессию
1. Прочитать `⚙️ Схема/Личность` · `⚙️ Схема/Контекст` · `⚙️ Схема/Операции`
2. Извлечь новое → `🧠 Вики/`
3. На ответ → `obsidian-sync` + git sync

## Структура
```
📦 Сырьё/     → статьи, заметки, файлы (неизменяемое)
🧠 Вики/      → знания (AI пишет)
⚙️ Схема/     → правила (вместе)
💬 Сессии/    → чаты (авто)
📋 Лог/       → действия (авто)
📥 Входящие/  → захват
🎯 Задачи/    → todo
```

## Стиль заметок
### ✅ Чекбоксы для задач
```markdown
- [x] Выполнено AI
- [ ] Ожидает выполнения
- [~] В процессе
```

### 📊 Красивые таблицы
```markdown
| Параметр | Значение | Статус |
|----------|----------|--------|
| GPU | RTX 3050 | ✅ Работает |
| DLSS | Quality | ✅ Настроено |
| RT | Off | ✅ Выключено |
```

### 🎨 Форматирование
- ✨ Эмодзи для наглядности
- 📝 Коротко и по делу
- 🔗 Связывать заметки ссылками
- 📊 Таблицы для структурированных данных
- ✅ Чекбоксы для отслеживания прогресса
