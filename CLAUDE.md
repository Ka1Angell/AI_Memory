# AI Memory Vault — Инструкции для AI

## О vault
Этот Obsidian vault — моя долговременная память. Твоя задача: читать, поддерживать и пополнять её.

## Структура

```
10 - Входящие/      # Быстрый захват (сброс идей, ссылок)
20 - Память/        # Структурированные знания (AI ведёт)
  ├── about.md      # Профиль, интересы, железо
  ├── working-context.md  # Текущий контекст
  ├── glossary.md   # Глоссарий терминов
  ├── решения/      # Важные решения
  ├── конфиги/      # Настройки (игр, софта)
  └── софт/         # Программы, утилиты
30 - Сессии/        # История разговоров (AI пишет)
40 - Действия/      # Лог действий AI
50 - Задачи/        # Задачи и todo
```

## Формат заметок
```yaml
---
type:         # profile | config | decision | software | session | task | inbox
status:       # active | done | archived
created:      YYYY-MM-DD
updated:      YYYY-MM-DD
---
```

## Что делать
- Первым делом читать: `about.md`, `working-context.md`, `glossary.md`
- Извлекать полезное из сессий в `20 - Память/`
- Обновлять `working-context.md` при смене контекста
- Записывать решения в `решения/`
- После каждой сессии: синхронизировать с GitHub
